VDU File Search

    Veteran's Discharge Upgrade File Search

    Veterans pursuing discharge upgrades often face long delays, fragmented communication, and limited visibility into their case progress. While official portals and service-specific reading rooms provide access to decisions and submission logs, they’re not designed for easy tracking or personal documentation.

    This tool streamlines the process. Simply enter your case number as provided by your board. The tool will scan the appropriate military reading room and return one of two results:

    File Not Found – Your case file has not yet been published.
    File Found – A direct link to the official .MIL site hosting your file.

    Built to reduce friction and restore control, this tool helps veterans monitor their upgrade status with clarity and precision.

    Case numbers vary by branch. Enter the full code as provided by your board.
